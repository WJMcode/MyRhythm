# MyRhythm

# Unity 2D Project

## 📌 프로젝트 개요
Unity 엔진과 C#을 학습하기 위한 2D 리듬 게임 프로젝트입니다.

## 🎯 학습 목표
- Unity 컴포넌트 시스템 및 GameObject 구조 이해
- C# 기본 문법 및 Unity API 학습
- 2D 게임 구조, 인터페이스, 입력 처리 구현
- Prefab, ScriptableObject 등 Unity 핵심 개념 습득

## 🛠️ 기술 스택
- **Engine**: Unity 6.2
- **Language**: C#
- **Platform**: Windows

## 📁 현재 구현 내용
- 씬 전환 입력 처리 구현
- 노래 선택 씬 구현
- FMOD를 통한 Audio 출력
